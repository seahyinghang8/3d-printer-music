# MIDI Library

Now with auto-track selection! The parser automatically finds the track with the highest frequency notes (usually the melody).

## Available MIDI Files

- Habanera (auto-selects best track)
- Jingle Bells (auto-selects best track)
- The Entertainer (Joplin)
- 2001: A Space Odyssey Theme
- Circus Theme Song
- Chopin - Waltz Opus 18
- Hey Jude (The Beatles)

## Using Playlists

You can now create playlists using `play_midi_playlist.py`! See `../playlist.yaml` for an example.

Features:
- Play multiple songs in sequence
- Trim songs with start/end time
- Auto-transpose and auto-track selection
- Shuffle mode
- Live editing (change settings between tracks)
- Ctrl+C to skip tracks
